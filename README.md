# Sistema de Servicios Estudiantiles 🎓

Una aplicación de gestión académica desarrollada en **C#** utilizando una **Arquitectura de 3 Capas** (Presentación, Lógica de Negocio y Acceso a Datos). Este sistema permite administrar de manera eficiente la información de estudiantes, asignaturas y directorios de profesores.

## 🚀 Arquitectura y Tecnologías

El proyecto está diseñado para ser escalable y mantenible, separando claramente las responsabilidades del sistema:

*   **Lenguaje:** C# / .NET
*   **Arquitectura:** Multicapa (Presentación, Negocio, Datos)
*   **Base de Datos:** SQL Server
*   **Despliegue / Cloud:** Microsoft Azure (App Service)
*   **Control de Versiones:** Git & GitHub
